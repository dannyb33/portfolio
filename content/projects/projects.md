# projects

* [**Policy Safe Prompt Library**](https://github.com/dannyb33/Policy-Safe-Prompt-Library)
    * Package for validating and templating LLM prompts for use in internal models
    * Features CLI and GUI to interact with the system hosted with Kubernetes

* [**Ego**](https://github.com/dannyb33/ego_v1)
    * Cloud-based social media app built with TypeScript with AWS Services.
    * Includes authentication, image storage, and profile customization

* [**Energy Package**](https://github.com/dannyb33/EnergyPackage)
    * Python package for simulating bitstring-based spin configurations.
    * Supports both exact and Monte Carlo simulation methods to compute average physical quantities.

* [**StopDropNRoll**](https://github.com/dannyb33/StopDropNRoll)
    * 24hr hackathon game to simulate fire safety drills.
    * Mobile game built in React Native that uses device motion sensors.

* [**France COVID Visualization**](https://dannyb33.github.io/FranceCOVIDVisualization/)
    * Data visualization project representing COVID statistics in France.
    * View different figures describing the French response to the COVID-19 pandemic.
